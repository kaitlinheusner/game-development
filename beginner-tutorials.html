Beginner Tutorials

There are many online tutorials that can help you learn how to make art or using different game engines, I will be listing a few here as well as explaining what they cover and teach.

1

How to Start Making Games with No Experience

This video covers where to start when making games. It even discusses the different aspects of game development like Game Engines and how to go about making your first game. Additionally, it gives recommendations for different programs that you can use to make the process easier. It also mentions how to get through the process when you feel stuck. This is great to get a general understanding of where to start.

2

The Unity Tutorial For Complete Beginners

This video is an in-depth beginner tutorial for Unity. It goes over the basic tools and systems that Unity has. Furthermore, it teaches you by giving you an example of a game that is similar to Flappy Bird that you can follow along with to understand how each feature works. This is a good tutorial to watch if you want to learn how to use Unity.

3

The Ultimate Pixel Art Tutorial

This is a video that goes over pixel art which is a useful skill if you are planning on making a 2D game. This video goes over the basics needed such as color selection, shading, and outlines. This is a useful tutorial to watch if you plan to work on the more artistic side of game development.

4

5 Steps To Gorgeous Game Art

This video goes over making both 2D and 3D art which allows for more flexibility in regard to what type of art style you would like to make for your game. These videos are made by game designers who work for Riot Games, a prominent company known for League of Legends and Valorant. These are useful videos to watch since you get information from professionals in the industry.
